Garansi Kualitas Sablon

Jika sablon rusak atau hasil cetakan tidak sesuai dengan desain awal yang telah disetujui, kami berikan garansi cetak ulang tanpa biaya tambahan. Kepuasan dan kepercayaanmu adalah prioritas utama kami.

CATATAN PENTING - FORMAT FILE DESAIN

Untuk hasil cetakan terbaik, kirimkan file desain dalam format PNG Transparan (resolusi minimal 300 DPI) atau format vektor CDR / AI. Hindari pengiriman file berformat JPG dengan latar belakang putih karena dapat memengaruhi ketajaman dan akurasi warna sablon.
